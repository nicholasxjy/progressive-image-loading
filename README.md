# NProgressiveImage

### How to use it

```html
<script src="../dist/NProgressiveImage.min.js"></script>
<script>
  window.onload = function() {
    new NProgressiveImage({
      container: '.content',
      items: '.nprogressive-image'
    })
  }
</script>
```
